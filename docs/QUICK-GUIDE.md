# Почтелье -- Руководство пользователя

Почтелье -- программа для создания и отправки HTML-писем по шаблонам.
Программа работает полностью на вашем компьютере и не требует установки дополнительного ПО.

---

## Первый запуск

### Что вам должны передать

Перед началом работы вы должны получить от ответственного за программу:

- файл `Pochtelye.exe` (Windows) или `Pochtelye.sh` (ALT Linux) -- сама программа
- файл `config.ini` -- настройки подключения к серверу
- файл `.lic` -- только если вы администратор

### Шаг 1. Положите файлы в одну папку

Создайте любую удобную папку (например, `C:\Pochtelye`) и скопируйте туда все полученные файлы. Важно, чтобы они находились рядом, а не в разных местах.

Обычному пользователю:

```text
Pochtelye.exe
config.ini
```

Администратору -- дополнительно:

```text
.lic
```

> Файл `.lic` -- это ваш ключ доступа к расширенным функциям. Без него программа
> работает в режиме обычного пользователя.

### Шаг 2. Запустите программу

**Windows:** дважды щёлкните по файлу `Pochtelye.exe`.

**ALT Linux:** откройте терминал в папке с файлами и выполните:

```bash
bash Pochtelye.sh
```

При первом запуске на ALT Linux программа установится в папку `~/Pochtelye/`
и создаст ярлык на рабочем столе. В следующий раз запускайте через ярлык.

### Шаг 3. Подождите первоначальную загрузку

При самом первом запуске программа загружает шаблоны и ресурсы с сервера в локальный
кеш на вашем компьютере. Это займёт несколько секунд или минут в зависимости от скорости
сети. Последующие запуски будут быстрее.

## Настройка подключения к Exchange

Чтобы отправлять письма прямо из программы, потребуется один раз ввести данные
вашего почтового ящика Exchange. Программа попросит об этом при первом обращении
к функции отправки.

Что нужно ввести:

- **Сервер** -- адрес почтового сервера вашей организации. Если не знаете --
  спросите у системного администратора (выглядит примерно так:
  `https://mail.company.ru/EWS/Exchange.asmx`)
- **Логин** -- ваш рабочий логин (обычно совпадает с логином от компьютера)
- **Пароль** -- ваш рабочий пароль

Данные сохраняются на вашем компьютере в зашифрованном виде и больше не запрашиваются.

## Что умеет программа

### Режим пользователя (по умолчанию)

Доступен всем. Открывается автоматически при отсутствии файла `.lic`.

- Выбирать готовые шаблоны писем из общей библиотеки
- Редактировать текст, изображения, кнопки в шаблоне
- Отправлять письмо через Exchange
- Создавать встречи в календаре Exchange

### Режим администратора

Доступен при наличии файла `.lic` рядом с программой.
Включает всё из режима пользователя, а также:

- Создавать и редактировать общие шаблоны для всех пользователей
- Добавлять категории шаблонов
- Обновлять общие ресурсы (иконки, баннеры и другие графические элементы)
- Переключаться в режим пользователя для проверки

## Если что-то пошло не так

### Программа не запускается

Проверьте, что файл `config.ini` лежит в той же папке, что и `Pochtelye.exe`.
Откройте `config.ini` в блокноте и убедитесь, что заполнена строка с адресом сервера:

```ini
[app]
network_path_win = \\server\share\email-builder   ; для Windows
```

```ini
[app]
network_path_smb = //server/share/email-builder   ; для ALT Linux
```

Если не знаете, что там должно быть -- обратитесь к тому, кто передал вам программу.

### При запуске появилось окно поиска сетевой папки

Это значит, что программа не смогла найти сервер автоматически. В окне есть три кнопки:

- **Проверить путь** -- нажмите, если вы вручную ввели или исправили путь в поле выше
- **Выполнить поиск** -- программа попробует найти сервер сама среди подключённых дисков
- **Создать новый** -- только для администраторов, создаёт новую пустую папку по указанному пути

После успешного поиска путь сохранится автоматически, и в следующий раз окно не появится.

### Нет доступа к серверу

Если сетевая папка сейчас недоступна (нет VPN, нет сети), программа всё равно
запустится и покажет ранее загруженные шаблоны из кеша. Отправка писем при этом
работает в обычном режиме.

### Письма не отправляются

Зайдите в настройки Exchange внутри программы и проверьте правильность сервера,
логина и пароля.

### Появилось сообщение о недействительном токене

Файл `.lic` не подходит к этой версии программы. Обратитесь к тому,
кто занимается обслуживанием программы, и попросите актуальный `.lic`.

### Шаблоны устарели или не обновляются

Когда администратор публикует новые шаблоны или ресурсы, при следующем запуске
программа предложит загрузить обновление. Если этого не произошло -- проверьте
доступность сетевой папки или запустите обновление вручную через меню программы.

---

## Обновление программы

### Windows

Замените файл `Pochtelye.exe` новым. Файл `config.ini` и `.lic` трогать не нужно.

### ALT Linux

Запустите новый установщик той же командой:

```bash
bash Pochtelye.sh
```

Программа сама определит, что нужно обновление, заменит только исполняемый файл
и сохранит все ваши настройки без изменений.
